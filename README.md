# rspec-daemon.vim

A Vim plugin for [rspec-daemon](https://github.com/asonas/rspec-daemon) gem.

## Requirements

- `nc` or `socat` to communicate with rspec-daemon.

## Installation

Use your favorite plugin manager, or use `:packadd` if you are a Vim8 user.
